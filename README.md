# AI Email Bot 🤖

Telegram бот для автоматической рассылки писем поставщикам.
Читает Excel файл, генерирует персональные письма через AI, отправляет через email.

## Как работает

1. Отправляешь Excel файл боту в Telegram
2. AI анализирует данные по каждому поставщику
3. Генерирует персональное письмо
4. Отправляет письмо на email поставщика

## Стек

- **n8n** — автоматизация workflow
- **Ollama + qwen3:4b** — локальный AI (бесплатно)
- **Yandex SMTP** — отправка писем
- **Docker Compose** — деплой

## Требования к серверу

- VPS 8GB RAM, 4 CPU, 50GB диск
- Docker + Docker Compose
- Linux (Ubuntu 22.04)
